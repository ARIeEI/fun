# Wheel Family
SVG를 이용하여 고래 가족이란 주제로 이미지를 만들었습니다.
낮과 밤이 바뀌는 것, 파도가 움직이는 것 등을 표현하여, 고래가족의 바다속 모습을 표현하고자 하였습니다.

- SVG animation

# Specification

* HTML5
* CSS
* SVG

# Challenges
* svg에 좌표를 찍어가며 고래의 형태와 파도의 자연스러운 형태를 그려내는게 쉽지 않은 작업이었습니다.
